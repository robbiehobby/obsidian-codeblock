Apply highlighting to inline code by adding the `{language}` attribute before the inline code. All built-in languages are supported. Highlight color can be changed by adding a color name parameter.

| Attribute                                | Result                            |
| ---------------------------------------- | --------------------------------- |
| <code class="is-loaded">{js}</code>      | `{js} console.log("hello world")` |
| <code class="is-loaded">{:red}</code>    | `{:red} console.log("1")`         |
| <code class="is-loaded">{:orange}</code> | `{:orange} console.log("1")`      |
| <code class="is-loaded">{:yellow}</code> | `{:yellow} console.log("1")`      |
| <code class="is-loaded">{:green}</code>  | `{:green} console.log("1")`       |
| <code class="is-loaded">{:cyan}</code>   | `{:cyan} console.log("1")`        |
| <code class="is-loaded">{:blue}</code>   | `{:blue} console.log("1")`        |
| <code class="is-loaded">{:purple}</code> | `{:purple} console.log("1")`      |
| <code class="is-loaded">{:pink}</code>   | `{:pink} console.log("1")`        |
