Display line numbers in code blocks.

In addition to the global setting, you can use the meta string specified in the opening fence to toggle line numbers.

**Enable:**

<div class="codeblock-help codeblock-line-hover codeblock-line-numbers">
  <code>
    <span class="line token comment">```js line-numbers</span>
    <span class="line"><span class="token property">console.log</span>(<span class="token string">"1"</span>)</span>
    <span class="line"><span class="token property">console.log</span>(<span class="token string">"2"</span>)</span>
    <span class="line"><span class="token property">console.log</span>(<span class="token string">"3"</span>)</span>
  </code>
</div>

**Disable:**

<div class="codeblock-help codeblock-line-hover">
  <code>
    <span class="line token comment">```js no-line-numbers</span>
    <span class="line"><span class="token property">console.log</span>(<span class="token string">"1"</span>)</span>
    <span class="line"><span class="token property">console.log</span>(<span class="token string">"2"</span>)</span>
    <span class="line"><span class="token property">console.log</span>(<span class="token string">"3"</span>)</span>
  </code>
</div>
