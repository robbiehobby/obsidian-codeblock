Apply syntax highlighting to inline code by adding the <span class="codeblock-help-inline">{language}</span> meta string before the inline code. All languages are supported.

<span class="codeblock-help-inline"><span class="token property">console.log</span>(<span class="token string">"1"</span>)</span> <span class="codeblock-help-inline">{js} console.log("1")</span>

You can also apply specific colors to plain text inline code.

<span class="codeblock-help-inline codeblock-inline-red">console.log("1")</span> <span class="codeblock-help-inline">{txt:red} console.log("1")</span>
