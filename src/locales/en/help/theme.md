The default theme automatically inherits the colors from your current Obsidian theme. Both light and dark themes are supported.

You can modify the default colors using the built-in CSS properties saved in a [CSS snippet](https://help.obsidian.md/snippets). The following CSS properties are currently supported.

<div class="codeblock-help codeblock-line-hover">
  <code>
    <span class="line"><span class="token keyword">body.obsidian-app</span> {</span>
    <span class="line">  <span class="token property">--code-background</span>: <span class="token string">black</span>;</span>
    <span class="line">  <span class="token property">--code-normal</span>: <span class="token string">white</span>;</span>
    <span class="line">  <span class="token property">--code-comment</span>: <span class="token string">gray</span>;</span>
    <span class="line">  <span class="token property">--code-function</span>: <span class="token string">yellow</span>;</span>
    <span class="line">  <span class="token property">--code-keyword</span>: <span class="token string">purple</span>;</span>
    <span class="line">  <span class="token property">--code-property</span>: <span class="token string">blue</span>;</span>
    <span class="line">  <span class="token property">--code-punctuation</span>: <span class="token string">gray</span>;</span>
    <span class="line">  <span class="token property">--code-string</span>: <span class="token string">green</span>;</span>
    <span class="line">  <span class="token property">--code-tag</span>: <span class="token string">red</span>;</span>
    <span class="line">}</span>
  </code>
</div>
